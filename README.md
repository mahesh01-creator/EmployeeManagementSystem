# Employee Management System 

###### *A full-stack web application to manage employee records with complete CRUD operations, built using Java, Spring MVC, Hibernate, and MySQL.*

###### 

###### 

#### **Features :-**



1. &#x20;Add New Employee
2. &#x20;Update Employee Details
3. &#x20;Delete Employee
4. &#x20;View All Employees
5. &#x20;Search Employees
6. &#x20;Organized Employee Data Management



### 

#### **Tech Stack :-**



1. &#x09;Backend : Java, Spring MVC
2. &#x09;ORM : Hibernate (JPA)
3. &#x09;Frontend : JSP, JSTL, HTML, CSS
4. &#x09;Database : MySQL
5. &#x09;Server : Apache Tomcat 10
6. &#x09;Build Tool : Maven

#### **How to Run the Project :-**



1\. Clone the repository :  git clone https://github.com/mahesh01-creator/EmployeeManagementSystem.git  

2\. Import as Maven Projec in Eclipse/IntelliJ

3\. Configure MySQL database in `persistence.xml`

4\. Run on Apache Tomcat Server

5\. Open in browser : http://localhost:8080/EmployeeManagementSystem/
